## Running Locally

Clone the project

```bash
git clone https://github.com/patnaikankit/terminal-portfolio.git
```

Go to the project directory

```bash
cd terminal-portfolio
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run dev
```
